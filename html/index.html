  2. Project Picster

    One project I had in college was to create a website that could do style transformations. It used keras and machine learning to apply classic images like the painting "Starry night" to style a user uploaded image. My work on it involved helping create a front-end UI for the project, which was a website. We used Flask to build the website. One interesting part of this project was that we used Docker to keep our python environments all the same, so I learned the very basics of using Docker in this project.

  3. Labyrinth of the Minotaur

    My first major project was a dungeon crawler written in java. It was the first time I used git and maven. Me and my teammates used Swing to draw the graphics, and found a library called kryonet to create a multiplayer server. The game is an infinite dungeon crawler, where you run around, fight enemies, pick up potions and keys, and attempt to make it as deep as you can. I helped code the game logic, and I wrote a maze generation algorithm using BFS.

  4. Chess Program

    In a class on C and C++ programming, we had the opportunity to do an assignment of our own design. I chose to create a chess client with SDL2. Although it doesn't support castling or enpassant, you can drag and drop the pieces, and it will exit at a checkmate. It's a couple thousand lines, and it was built in only a couple of weeks. In this project I learned about SDL2, and continued on my skills with Makefiles and C++.
